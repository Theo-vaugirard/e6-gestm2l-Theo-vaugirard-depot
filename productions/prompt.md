
# Amélioration de l'interface utilisateur pour la création de salles

IA utiliser : Github Copilot chat

## Prompt 1 :

Lors de la création d'une nouvelle salle, une fenêtre dialogue s'affiche pour permettre à l'utilisateur
de saisir les détails de la salle.

On vous demande de rendre L'interface utilisateur plus agréable en mettant en place des espacements entre
les éléments et une icône (que je vais fournir) pour la fenêtre de dialogue.


Un bouton permettant de fermer la fenêtre de dialogue et d'annuler la création de la salle doit également être
ajouté pour améliorer l'exprerience utilisateur. De plus, un menu déroulant doit être mise en place pour permettre
à l'utilisateur de sélectionner un bâtiment existant.

## Prompt 3 :

Par la suite, services et équipements doivent être choisis à partir de listes déroulantes pour faciliter
la séléction et éviter les erreurs de saisie. Cette liste déroulante sera alimentée par les données existantes dans
de données pour garantir que les options disponibles sont à jour et pertinentes.

En vue de cette fonctionnalité futur, on vous demande d'écrire et de valider le fichier de migration qui permettra
de mettre à jour la base de données.

Vous devez rédiger une courte documentation expliquant les étapes nécessaire pour effectuer cette migration,
y compris les commandes à executer et les précautions à prendre pour éviter toute perte de données.
